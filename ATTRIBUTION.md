## License-Safe Usage

TrendPilot AI may include:

- Original prompts
- Original report templates
- Original workflow concepts
- Original architecture documentation
- Original scoring models
- Original compliance checklists
- Public links to third-party projects
- Educational notes about integration ideas

TrendPilot AI should not include:

- Third-party source code copied without permission
- Third-party workflow exports copied without permission
- Removed or modified third-party license files
- Third-party branding used in a misleading way

- Repackaged external tools presented as original TrendPilot AI code
